# magic-drawing-canvas
Magic Drawing Canvas 🎨 - An interactive web-based drawing application where you can draw, change colors, adjust brush sizes, and save your artwork as PNG. Built with vanilla HTML, CSS, and JavaScript. It lets you unleash your creativity with a beautiful interface and smooth drawing experience.

[FREE ACCESS to Magic Drawing Canvas](https://tomato-automation.github.io/magic-drawing-canvas)

## ✨ Features

- 🖌️ **Free Drawing** - Click and drag to draw anything you imagine
- 🎨 **Color Picker** - Choose from millions of colors
- 📏 **Adjustable Brush Size** - From 1px to 50px
- 🧹 **Clear Canvas** - Start over with one click
- 💾 **Save as PNG** - Download your artwork instantly
- 📱 **Mobile Friendly** - Works on touch screens
- ✨ **Smooth Experience** - Optimized drawing performance
